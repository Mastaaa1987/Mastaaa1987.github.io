1. Installiere Termux
* Oeffne den Google Playstore & suche nach Termux + Installiere Termux.
Screenshots




2. Starte und bereite Termux vor
* Oeffne die App Termux & gib folgende Befehle ein:
(1 Befehl pro Zeile. $ steht fuer eine Befehlszeile & gehoert nicht zum Befehl dazu!)
$ apt update
$ pkg upgrade
$ apt -y upgrade
(Alle fragen einfach mit [OK/ENTER] bestaetigen.)
$ apt install python-pip
(Die frage wieder mit [OK/ENTER] bestaetigen.)
$ pip install vxparser
(Jetzt einfach nur warten bis meine App installiert ist ...)
Screenshots














3. Start und benutze VxParser Api
* Meine Server Api wird nun ueber diesen Befehl gestartet:
$ vxparser
Nachdem FastApi Server gestartet ist, zeigt sich ein Menu, welches einfach ueber die TV Fernbedinung zu steuern ist.
